Visual Studio Code Tips & Tricks by Hasin Hayder

  • Material Icon Theme
  • One Dark Theme
  • "ctrl+d" to duplicate line or selection
  • "alt+up/down arrow" to move line up or down
  • To enable minimap => settings to minimap
  • To go particular line quickly: ctrl+shift+p then (>goto line) {shortcut: ctrl+g} go to line number
  • To hide side bar ctrl+b
  • To split window, right click then split,
  • To go active tab just press ctrl + tab no
  • I can create nested folder easily in navigation panel by putting folder name/new name/ new folder or file name
  • For multiple editing I can use multi cursor by pressing atl+click
  • To open different file into different tab, not in same tab who remove the existing tab change the setting from file=>preference=>Workbench Editor: Enable Preview (Dsiable this)
  • ctrl+tab to quickly switch recent tab
  • To close all the tab at a time ctrl+k+w
  • Terminal shortcut ctrl+`
  • to go all the symbol of a file ctrl+shift+O
  • To formate the codes alt+shift+F (have to be a formatter extension installed)
